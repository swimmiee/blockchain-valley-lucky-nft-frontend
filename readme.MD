# Blochchain Valley Lucky NFT 세션

## Install
```
git clone https://github.com/swimmiee/blockchain-valley-lucky-nft-frontend
npm install
```

## Prerequisites
0. Metamask 설치
1. Faucet 받기 https://baobab.wallet.klaytn.foundation/

## Update
```
git pull
```
